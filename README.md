## Getting Started


### Prerequisites
* Python3.7
  ```sh
  sudo apt-get python3.7
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/steven0323/GCP-Carrefour.git
   ```
2. Install pip packages
   ```sh
   sudo apt-get pip
   ```
